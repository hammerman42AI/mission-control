# Aegis Mission Control

**The "Jarvis" Vision for OpenClaw**

A standalone, local-first SecOps dashboard and "Life OS" command center.

## Architecture
- **Orchestrator:** Gandalf (The White Wizard)
- **Council members:** Celebrimbor (Dev), Samwise (Coord), Legolas (Sentinel), Elrond (Librarian)
- **Stack:** Node.js, Socket.io, Gridstack.js, Chart.js

## Features
- **Live System Vitals:** Real-time hardware telemetry from Mac Studio.
- **Ring of Power:** Active token burn tracking across all models.
- **Book of Mazarbul:** Live-tailed audit log of all system skill executions.
- **Mission Pipeline:** Visual stage tracking (Consultation -> Delivery).

## Setup
```bash
npm install
node server.mjs
```
Open [http://localhost:8888](http://localhost:8888)
